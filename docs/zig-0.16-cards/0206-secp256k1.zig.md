# Migration Card: std/crypto/pcurves/tests/secp256k1.zig

## 1) Concept

This is a test file for the secp256k1 elliptic curve implementation in Zig's standard crypto library. It contains comprehensive test cases that validate the core cryptographic operations including ECDH key exchange, point operations, coordinate transformations, SEC1 encoding/decoding, and scalar operations. The tests verify mathematical correctness through test vectors and property-based testing.

Key components tested include:
- Elliptic curve point multiplication (both regular and public variants)
- Affine coordinate conversions and serialization
- SEC1 compressed/uncompressed encoding formats
- Scalar operations (inversion, parity checking)
- Edge cases like identity element handling and non-canonical encodings

## 2) The 0.11 vs 0.16 Diff

This test file demonstrates several migration patterns from Zig 0.11 to 0.16:

**Error Handling Changes:**
- Explicit error types instead of generic errors (e.g., `error.IdentityElement`, `error.NonCanonical`, `error.InvalidEncoding`)
- Error handling via `try` and `expectError` with specific error sets

**API Structure Changes:**
- Point creation uses factory functions like `Secp256k1.fromSerializedAffineCoordinates()` and `Secp256k1.fromSec1()` instead of direct struct initialization
- Endianness specification required in operations (`.little`, `.big` parameters)
- Method chaining patterns with explicit error propagation

**Key Signature Changes Observed:**
- `Secp256k1.fromSerializedAffineCoordinates(xs, ys, .big)` - explicit endianness
- `point.mul(scalar, .little)` vs `point.mulPublic(scalar, .little)` - separate public/private multiplication
- `Secp256k1.mulDoubleBasePublic()` - multi-point multiplication with public verification

## 3) The Golden Snippet

```zig
const std = @import("std");
const Secp256k1 = @import("../secp256k1.zig").Secp256k1;

test "secp256k1 ECDH key exchange" {
    const dha = Secp256k1.scalar.random(.little);
    const dhb = Secp256k1.scalar.random(.little);
    const dhA = try Secp256k1.basePoint.mul(dha, .little);
    const dhB = try Secp256k1.basePoint.mul(dhb, .little);
    const shareda = try dhA.mul(dhb, .little);
    const sharedb = try dhB.mul(dha, .little);
    try std.testing.expect(shareda.equivalent(sharedb));
}
```
