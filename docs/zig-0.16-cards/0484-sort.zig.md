# Migration Card: std/sort.zig

## 1) Concept

This file provides a comprehensive collection of sorting algorithms and search utilities for the Zig standard library. It contains both stable and unstable in-place sorting algorithms including insertion sort, heap sort, block sort, and pattern-defeating quicksort (PDQ). The module also provides binary search variants, min/max finding utilities, and sortedness checking.

Key components include:
- Multiple sorting algorithms with consistent interfaces
- Context-based and direct comparison function patterns
- Binary search operations (lowerBound, upperBound, partitionPoint, equalRange)
- Min/max finding utilities with arg variants
- Ascending/descending comparator generators

## 2) The 0.11 vs 0.16 Diff

**No significant migration changes detected.** This module maintains consistent patterns across versions:

- **No allocator requirements**: All sorting algorithms are in-place and don't require memory allocation
- **Consistent context pattern**: Functions accept generic context parameters for comparison logic
- **Stable function signatures**: Public APIs follow the same patterns as Zig 0.11
- **Generic comparison functions**: Use `comptime` function parameters for type-safe comparisons

The API structure remains unchanged with:
- Direct sorting functions (`insertion`, `heap`, `block`, `pdq`)
- Context-based variants (`insertionContext`, `heapContext`, `pdqContext`)
- Search operations (`binarySearch`, `lowerBound`, `upperBound`, `partitionPoint`, `equalRange`)
- Utility functions (`asc`, `desc`, `min`, `max`, `argMin`, `argMax`, `isSorted`)

## 3) The Golden Snippet

```zig
const std = @import("std");
const sort = std.sort;

// Sort a slice of integers in ascending order
var numbers = [_]i32{ 3, 1, 4, 1, 5, 9, 2, 6 };
sort.insertion(i32, &numbers, {}, sort.asc(i32));

// Use binary search to find an element
const found_index = sort.binarySearch(i32, &numbers, @as(i32, 4), struct {
    fn order(ctx: i32, item: i32) std.math.Order {
        return std.math.order(ctx, item);
    }
}.order);

// Find minimum value
const min_val = sort.min(i32, &numbers, {}, sort.asc(i32));
```

## 4) Dependencies

- `std.mem` - For swap operations and slice comparisons
- `std.math` - For ordering comparisons and overflow-safe arithmetic
- `std.debug` - For runtime assertions
- `std.testing` - For test utilities (test-only)

**Note**: The module imports sub-modules `sort/block.zig` and `sort/pdq.zig` for additional sorting algorithms.
