# Migration Card: `std/math/complex/arg.zig`

## 1) Concept
This file implements the `arg` function for complex numbers, which returns the angular component (phase angle) of a complex number in radians. The function calculates the angle using the `atan2` function from the standard math library, taking the imaginary and real components of the complex number.

The key component is a single public function `arg` that operates on complex numbers of any type, using Zig's generic type system to handle different floating-point precisions while maintaining type safety.

## 2) The 0.11 vs 0.16 Diff
- **Function Signature**: The `arg` function uses modern Zig generic syntax with `anytype` parameter and `@TypeOf` for return type inference
- **Complex Number Construction**: Uses `Complex(T).init()` factory pattern rather than direct struct initialization
- **No Allocator Required**: Pure mathematical function with no memory allocation
- **Error Handling**: Uses testing framework with `try` syntax for error propagation in tests
- **API Structure**: Standalone mathematical function rather than method on complex type
